Setting up your Build

So if you have made it this far then you should have all of your parts now. Before we get to putting it all together there are a few extra tools we need. First off we are going to need a large and open workspace over a hardwood surface. It is important to stand over a non carpeted surface to avoid unintentional static to damage sensitive parts like the CPU and GPU.

Secondly you will absolutely need a Phillips #2 Screwdriver and if you are using an M.2 storage, then a Phillips #0 Screwdriver.(A magnetic screwdriver will help prevent losing screws in tight places)

Lastly are some optional items that the build can be conducted without but might be good to have some on hand. These items include Zipties and a set of pliers to tighten the Zipties which will help with cable management; a static bracelet to ground you and ensure no damage is done to static sensitive parts; and extra thermal paste in case the CPU cooling you have does not include it.
